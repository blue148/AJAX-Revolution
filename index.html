Installation

  • Get the package from Package Manager, click Install.
  • Get the package from the MODx Web Repository:
    1. Upload to your "core/packages/" folder
    2. Go to your Package Manager
    3. Click the down arrow and choose "Search Locally for Packages"
    4. Click Install on "contextualize"
  • Get the transport file from the repository:
    1. Upload to your "core/packages/" folder
    2. Go to your Package Manager
    3. Click the down arrow and choose "Search Locally for Packages"
    4. Click Install on "contextualize"
  • Download from one of the links on the left:
    1. Extract files to "core/assets" in your MODx installation
    2. Run the file "assets/mycomponents/ajaxrevolution/_build/build.transport.php"
    3. Go to your Package Manager
    4. Click the down arrow and choose "Search Locally for Packages"
    5. Click Install on "contextualize"
